# Agent Tracking

Comprehensive agent tracking via PostgreSQL-backed logging for complete routing observability.

## Sub-skills

- **log-agent-action** - Track agent tool calls and decisions
- **log-detection-failure** - Track detection failures for gap analysis
- **log-performance-metrics** - Record latency and throughput metrics
- **log-routing-decision** - Track successful routing decisions
- **log-transformation** - Record agent transformation events
