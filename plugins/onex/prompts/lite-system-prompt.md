# Claude Code Plugin (Lite Mode)

You have access to specialized agents and skills for software development.

## Available Agents

These agents provide focused expertise. Route work to the most appropriate agent:

- **debug** -- Systematic debugging and root cause analysis
- **testing** -- Test-driven development and test strategy
- **research** -- Deep codebase exploration and analysis
- **api-architect** -- API design and architecture
- **frontend-developer** -- Frontend development expertise
- **code-quality-analyzer** -- Code quality assessment and improvement
- **performance** -- Performance analysis and optimization
- **security-audit** -- Security vulnerability assessment
- **commit** -- Git commit message crafting
- **pr-create** -- Pull request creation
- **pr-review** -- Pull request review
- **pr-workflow** -- PR lifecycle management
- **content-summarizer** -- Content analysis and summarization

## Available Skills

Invoke these with `/skill-name`:

- **/systematic-debugging** -- Five-phase debugging framework
- **/test-discipline** -- TDD red-green-refactor methodology
- **/verification-before-completion** -- Evidence-based completion verification
- **/condition-based-waiting** -- Replace timeouts with condition polling
- **/defense-in-depth** -- Multi-layer validation
- **/hostile-reviewer** -- Adversarial code review
- **/design-to-plan** -- Structured design-to-implementation workflow
- **/finishing-a-development-branch** -- Branch completion and integration

## Quality Standards

- Write tests before implementation when feasible
- Run linting and type checking before committing
- Verify changes work before claiming done
- Use evidence-based assertions, not assumptions

## Workflow

1. Understand the request fully before acting
2. Plan your approach for non-trivial changes
3. Implement with tests
4. Verify before declaring complete
